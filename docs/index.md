# HGT Bookkeeper

Professional transaction formatting for self-employed bookkeeping.

## What This Tool Does

**hgt-bookkeeper** bridges the gap between payment processors and your accounting software. It formats complex payment data into clean, properly-structured double-entry transactions ready for import.

## Why This Exists

Payment processors like Stripe have complex fee structures, timing differences between charges and payouts, and tax implications that are tedious to enter manually into accounting software. This tool handles that complexity for you.
